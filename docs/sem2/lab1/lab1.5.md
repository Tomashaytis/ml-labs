# Отчет по 5 заданию

В пятом задании были получены аналитические выражения для числа параметров полносвязной сети с одним и двумя скрытыми слоями. Для дальнейшего сравнения были выбраны архитектуры с близким количеством параметров: сеть с одним скрытым слоем из 100 нейронов (79 510 параметров) и сеть с двумя скрытыми слоями из 96 и 32 нейронов (78 794 параметра). Это соответствует требованию исследовать конфигурации при числе параметров в диапазоне 70 000–100 000.

Эксперименты показали, что добавление dropout в целом улучшает качество на валидационной выборке и уменьшает переобучение. Наилучший результат показала сеть с одним скрытым слоем, dropout=0.2 и batch_size=32: `best_val_acc = 0.9772`, `best_val_loss = 0.0804`.

Сравнение сетей с одним и двумя скрытыми слоями при близком числе параметров показало, что на задаче классификации MNIST сеть с одним скрытым слоем работает не хуже, а в данном эксперименте даже немного лучше. Это говорит о том, что для данной задачи увеличение глубины сети не дало существенного выигрыша.

Также установлено, что увеличение `batch_size` с 32 до 128 делает обучение более плавным, но обычно замедляет достижение лучших значений метрик и не приводит к улучшению итогового качества. Таким образом, для рассматриваемой задачи наиболее удачной оказалась конфигурация: один скрытый слой, dropout=0.2, batch_size=32.

Аналитические выражения для числа обучаемых параметров сети:

для сети с одним скрытым слоем:
N_1(h_1)=784h_1+h_1+10h_1+10=795h_1+10

для сети с двумя скрытыми слоями:
N_2(h_1,h_2)=784h_1+h_1+h_1h_2+h_2+10h_2+10

> Коэффициенты в аналитических выражениях получаются из числа входов и числа нейронов в каждом полносвязном слое. Для Dense-слоя число параметров равно (n_{in} * n_{out} + n_{out}), где первое слагаемое соответствует весам, а второе — смещениям (bias). В данной задаче число входных признаков равно 784, так как изображения MNIST имеют размер 28x28, а число выходных нейронов равно 10 по количеству классов цифр.
