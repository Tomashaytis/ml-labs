************************************************************************************************

Если `torch.cuda.is_available()` возвращает False, хотя на ПК есть GPU
Это частая проблема. Обычно дело не в железе или драйверах, а в том, что установлена версия PyTorch без поддержки CUDA.
Выполняйте следующие шаги:

Шаг 1: Диагностика 

Запустите диагностический скрипт cda_test.py в вашем Python окружении (или прямо в файле проекта). 
**Анализ результатов:**

| Что видим в выводе | Вероятная причина | Что делать |
|-------------------|-------------------|------------|
| `nvidia-smi` не работает | Драйвер NVIDIA не установлен или сломан | **Способ 1: Установить/переустановить драйвер** |
| В версии PyTorch есть `+cpu` или список архитектур пуст `[]` | Установлена CPU-версия PyTorch | **Способ 2: Переустановить PyTorch с CUDA** |
| Драйвер есть, но ошибка про "слишком старый драйвер" | Драйвер устарел для этой версии PyTorch | **Способ 3: Обновить драйвер NVIDIA** |
| Всё проверили, но не работает | Конфликт окружений или антивирус | **Способ 4: Создать чистое окружение** |

---

### Шаг 2: Выберите нужный способ

#### Способ 1: Установка/переустановка драйвера NVIDIA

Если `nvidia-smi` не работает:
1. Скачайте драйвер с [официального сайта NVIDIA](https://www.nvidia.ru/Download/index.aspx?lang=ru)
2. Выберите свою видеокарту и ОС
3. При установке выберите "Выборочная установка" → "Выполнить чистую установку"
4. Перезагрузите компьютер

#### Способ 2: Переустановка PyTorch с поддержкой CUDA

Если у вас CPU-версия (список архитектур пуст):

1. **Полностью удалите старую версию** (в терминале, НЕ в Python):
```bash
pip uninstall torch torchvision torchaudio -y
```

2. **Установите правильную версию**:
   - Зайдите на [официальный сайт PyTorch](https://pytorch.org/)
   - Выберите: Stable → Windows → Conda/Pip → **CUDA 11.8 или 12.1**
   - Скопируйте и выполните предложенную команду

   **Для Conda (рекомендуется)**:
```bash
conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia
pip3 install torch torchvision --index-url https://download.pytorch.org/whl/cu126
```

   **Для Pip**:
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

#### Способ 3: Обновление драйвера NVIDIA

Если ошибка про устаревший драйвер:
- Скачайте и установите **последнюю версию** драйвера с сайта NVIDIA
- Новые драйверы поддерживают старые версии CUDA, так что обновляйтесь смело

#### Способ 4: Создание чистого окружения (если ничего не помогло)

Это самый надежный способ, если окружение "засорено":

```bash
# Создаем новое окружение
conda create -n pytorch_gpu python=3.10 -y

# Активируем его
conda activate pytorch_gpu

# Устанавливаем PyTorch с CUDA
conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia
```

**Важно!** После этого в вашей IDE (PyCharm, VSCode и т.д.) нужно выбрать новый интерпретатор `pytorch_gpu`.

---

### Проверка результата

После выполнения любого из способов снова запустите диагностический скрипт. Если всё хорошо, вы увидите:

```
CUDA доступна: True
Поддерживаемые CUDA архитектуры: ['sm_75', 'sm_80', ...]  # не пустой список!
```
